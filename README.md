# LegacyBuilt Command OS™ v3.2.9 (Desktop PWA)

Installable PWA optimized for desktop/laptop with offline support.

## Features
- Sidebar navigation (robust event delegation)
- Dashboard: Quick Income (with $ totals) + Quick Tasks
- Lead Tracker -> Calendar (Month grid + Upcoming)
- Mindset Vault (affirmations + journal)
- Brand Library
- Content Vault: Post Generator + Caption Generator + Hashtag Generator + Library
- Planner (tasks with due dates, notes, export)
- Offline cache via Service Worker
- Local persistence via localStorage

## Deploy to GitHub Pages
1. Create a new repo on GitHub (public).
2. Upload the contents of this folder (`index.html`, `styles.css`, `app.js`, `manifest.webmanifest`, `service-worker.js`, `assets/`).
3. In the repo: **Settings → Pages → Build and deployment → Source**: choose **Deploy from a branch**.
4. Select **main** branch, **root** (/), and **Save**.
5. Wait for the page to build. Your app will be available at `https://<your-username>.github.io/<repo-name>/`.
6. In Chrome/Edge: open the site and **Install app** from the browser menu. Works offline after first load.
